# SVG Symbols Collection

A comprehensive collection of outdoor recreation and park symbols in SVG format. Most have been derived from the [NPS Library](https://github.com/nationalparkservice/symbol-library/?tab=readme-ov-file). 
All icons are in **SVG format** and most are designed for consistency in size and style.
